## Changes
- Chat input and Send button

  - Send button now uses `UI_COLORS.buttonPrimaryBg` with `onMouseEnter` / `onMouseLeave` hover handlers.

- Tutorial buttons
  - Replaced Tailwind `bg-blue-600` / `hover:bg-blue-700` styles with inline styles using `UI_COLORS.buttonPrimaryBg` and hover handler `UI_COLORS.buttonPrimaryBgHover`.
  - Skip button uses `UI_COLORS.buttonTertiaryBg` / `UI_COLORS.buttonTertiaryBgHover`.
  - File: `frontend/src/components/TutorialControls.jsx` and tutorial instances in `ChatPanel.jsx` updated.

- Sparkles icon (assistant messages)
  - Replaced the `✨` emoji spans in `ChatPanel.jsx` with the filled Heroicons "sparkles" SVG.
  - Icon color is driven by `UI_COLORS.chatIconColor` so it follows your theme.
  - All chat message types that previously showed the sparkle emoji now render the SVG.

## Rationale
- Centralizing colors in `UI_COLORS` reduces duplication and makes theme adjustments easier.
- Using inline styles + small event handlers preserves hover/focus states while allowing colors to come from JS constants.
- Replacing emoji with SVG icons provides consistent sizing and color control.
